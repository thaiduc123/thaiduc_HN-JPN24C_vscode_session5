5. Groups of selectors

A coma-sperated list of selectors represents the union of all element selected by each of the individul selectors in the list. (A comma is U+002C.) for example, in CSS when all of the selectors share the same declaration, they may be grouped into a coma-seperated list.

CSS example:

In this example, we condense three rules with identical declaration into one. Thus,

h1 { font-family: sans-serif; }
h2 { font-family: sans-serif; }
h3 { font-family: sans-serif; }

is equivalent to:

h1, h2, h3 { font-family: sans-serif; }
